React PhoneBook API + React Web Build
==================================

Simple React PhoneBook Application

Requirement
---------------
```
Node >= 7.10.1
MongoDB 4 and above
```

Getting Started
---------------

```sh
# Install dependencies
npm install

# Start server
npm run server

# Start client
npm run client

# Start both server and client
npm start

# Install & Build React Production
npm run build
```
